# Contact Form Application

A full-stack Contact Form application built using **React**, **Node.js**, **Express**, and **MongoDB Atlas**.  
The application allows users to submit contact details, which are securely stored in a MongoDB database.

## Features

- User-friendly contact form
- Client-side validation (name, email, phone)
- RESTful API using Express
- MongoDB Atlas integration
- Environment variable based configuration
- Secure handling of sensitive credentials
- Clean and modular code structure

## Tech Stack

**Frontend**
- React
- JavaScript
- HTML & CSS

**Backend**
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
